Einen Ordner im aktuellen Verzeichnis anlegen

```powershell
New-Item -Path MeinNeuerOrdner -ItemType Directory
```

Eine Datei im Ordner C:\Users\Benutzer anlegen

```powershell
New-Item -Path C:\Users\Benutzer\MeineNeueDatei -ItemType File
```
