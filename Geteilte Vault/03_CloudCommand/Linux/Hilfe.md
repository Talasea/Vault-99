Hilfe für ein Kommando anzeigen lassen

```bash
# commando -h oder --help (beides versuchen falls eines nicht verfügbar ist)
apt -h
apt --help
```

Manual page für `apt` öffnen (meistens mehr Info, existiert nicht für alle Kommandos)

```bash
man apt
```
