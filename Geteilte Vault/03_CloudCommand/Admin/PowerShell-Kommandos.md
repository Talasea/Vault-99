# Dateisystem

Eingabe des Benutzers erfragen und in der Variablen `$alter` speichern

```powershell
$alter = Read-Host -Prompt "Bitte gib Dein Alter an"
```

Datei umbenennen

```powershell
mv $alterName $neuerName
mv .\hello-world.txt .\hello-world.ps1
```
