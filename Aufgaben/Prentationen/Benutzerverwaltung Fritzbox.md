**Warum ist die Benutzerverwaltung wichtig?**

- **Sicherheit:** Sie können verhindern, dass unbefugte Personen auf Ihre FRITZ!Box-Einstellungen zugreifen und Änderungen vornehmen, die Ihre Internetverbindung oder Ihr Heimnetzwerk gefährden könnten.
- **Kontrolle:** Sie behalten die Kontrolle darüber, wer welche Funktionen nutzen darf. Zum Beispiel können Sie den Zugriff auf das NAS-Laufwerk oder VPN-Verbindungen beschränken.
- **Individuelle Berechtigungen:** Sie können jedem Benutzer individuelle Rechte zuweisen, je nachdem, welche Aufgaben er oder sie übernehmen soll.

**Zugriff auf die Benutzerverwaltung der FRITZ!Box**

Um zur Benutzerverwaltung zu gelangen, gehen Sie wie folgt vor:

1. **Öffnen Sie Ihren Internetbrowser.** (z.B. Chrome, Firefox, Edge, Safari).
2. **Geben Sie in die Adresszeile `fritz.box` ein und drücken Sie die Eingabetaste.** Alternativ können Sie auch die IP-Adresse Ihrer FRITZ!Box eingeben (in der Regel `192.168.178.1` oder `169.254.1.1`).
3. **Melden Sie sich an der FRITZ!Box-Benutzeroberfläche an.** Geben Sie Ihr FRITZ!Box-Kennwort ein, falls Sie dazu aufgefordert werden. Wenn Sie das Kennwort vergessen haben, müssen Sie möglicherweise die FRITZ!Box auf Werkseinstellungen zurücksetzen.
4. **Klicken Sie im Menü links auf "System".**
5. **Klicken Sie im Untermenü "System" auf "FRITZ!Box-Benutzer".**

Sie befinden sich nun in der Übersicht der FRITZ!Box-Benutzerverwaltung. Hier können Sie verschiedene Aktionen durchführen.

**Funktionen der Benutzerverwaltung**

In der Benutzerverwaltung der FRITZ!Box finden Sie folgende Optionen:

- **Benutzer hinzufügen:** Hier können Sie neue Benutzerkonten erstellen.
- **Benutzer bearbeiten:** Sie können bestehende Benutzerkonten ändern, z. B. den Namen, das Kennwort und die Berechtigungen.
- **Benutzer löschen:** Sie können Benutzerkonten entfernen, wenn diese nicht mehr benötigt werden.

**Wichtige Hinweise zur Benutzerverwaltung:**

- **Sichere Kennwörter:** Verwenden Sie für alle Benutzer sichere Kennwörter, die schwer zu erraten sind.
- **Minimale Rechtevergabe:** Vergeben Sie Benutzern nur die Rechte, die sie wirklich benötigen. Je weniger Rechte ein Benutzer hat, desto geringer ist das Risiko, dass er versehentlich oder absichtlich Schaden anrichten kann.
- **Regelmäßige Überprüfung:** Überprüfen Sie regelmäßig die Benutzerverwaltung und entfernen Sie Benutzerkonten, die nicht mehr benötigt werden.
- **Administrator-Kennwort:** Vergessen Sie nicht Ihr eigenes FRITZ!Box-Administrator-Kennwort! Dieses Kennwort benötigen Sie, um auf alle Einstellungen zuzugreifen und die Benutzerverwaltung zu konfigurieren.
