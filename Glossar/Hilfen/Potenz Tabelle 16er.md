

Eine 16ner Potenz Tabelle zeigt die Ergebnisse von 16 hoch n, wobei n eine natürliche Zahl ist. Hier sind einige Beispiele:

- 16^1 = 16
- 16^2 = 256
- 16^3 = 4096
- 16^4 = 65536
- 16^5 = 1048576
- 16^6 = 16777216
- 16^7 = 268435456
- 16^8 = 4294967296
- 16^9 = 68719476736
- 16^10 = 1099511627776
